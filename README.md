# Интернет-магазин GymBoss | Full-stack учебный проект

**Стек:** PHP, MySQL, JavaScript, HTML/CSS
**Статус:** Основной функционал реализован. Проект в активной разработке.

## Функционал

- **Frontend:** Адаптивный интерфейс на чистом JS. Работа с корзиной, каталогом товаров, формами.
- **Backend:** PHP с авторизацией пользователей.
- **База данных:** MySQL. Спроектирована и реализована схема для товаров, заказов, пользователей.
- **Интеграции с внешними API:**
    - **Платежи:** ЮKassa (прием онлайн-оплат).
    - **Доставка:** Яндекс Карты (выбор пункта выдачи или адресов доставки).
    - **Валидация:** DaData (подсказки адресов).
    - **Уведомления:** SMSC (отправка SMS с кодом подверждения телефона).

## Цель проекта

Целью было создание с нуля полнофункционального веб-приложения для отработки навыков full-stack разработки, работы с внешними сервисами и проектирования БД.
